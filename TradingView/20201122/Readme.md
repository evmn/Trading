以后ETH最好在557,568这样的位置开空单，XRP最好在0.39,0.48这样的位置开空。不要急着入场，也不要设置太小的止损，开完单就不再去关注。我想有可能ETH在550是个硬阻力，但是涨势太强，所以就给涨了上去，以后要考虑这种飞速上涨和缓慢上涨的情形。
